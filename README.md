# WanderCoin - Travel Planner

A simple and intuitive travel planner application that helps you plan your perfect trip within your budget.

## Description

WanderCoin is a single-page application built with HTML, Tailwind CSS, and vanilla JavaScript. It allows users to input their travel destination, budget, duration, and number of travelers to generate customized trip plans. The application offers different travel styles such as Backpacker, Balanced, and Comfort, each with a different budget allocation for transport, stay, food, and activities.

## Features

- **Budget-Based Planning:** Plan your trip based on your total budget.
- **Multiple Travel Styles:** Choose from three different travel styles: Backpacker, Balanced, and Comfort.
- **Dynamic Budget Allocation:** The application automatically allocates your budget to different categories like transport, stay, food, and activities based on your chosen travel style. You can also customize the allocation.
- **Auto-Generated Itinerary:** Get a day-by-day itinerary with suggested activities.
- **Expense Tracker:** Keep track of your expenses during the trip.
- **Light & Dark Mode:** Switch between light and dark themes for a better user experience.
- **Responsive Design:** The application is designed to work on both desktop and mobile devices.

## How to Use

1.  Clone the repository or download the `index.html` file.
2.  Open the `index.html` file in your web browser.
3.  Enter your destination, budget, trip duration, and the number of travelers.
4.  Choose a travel style that suits you.
5.  Explore the generated itinerary, recommendations, and start tracking your expenses.
